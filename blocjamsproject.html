Bloc Jams About the project This is a student project done through Bloc.io's curriculum. This project's LOE's were close to 80 hours. This project contained learning centered around HTML5, CSS3, jQuery, JavaScript, DOM Scripting, DOM Events, and AngularJS. Bloc Jams is an interactive music player, similar to Spotify. Design aspects were provided by Bloc, to ensure the curriculum focused on JavaScript, DOM events, and Functionality.
Part One
The first part of this project revolved around HTML and CSS aspects. The design elements were chosen for us, but the curriculum was designed around responsive web development, animation, z-indexes, layout basics, and fundamental programming concepts using JavaScript.
Part Two
The second part of the project dove deeper into refactoring vanilla Javascript with jQuery. This section focused heavily on event handlers, event bubbling, event listening, hierachys, and debugging.
